## â“ Troubleshooting

### "Invalid API Key" Error
- Make sure you copied the **entire key** (starts with `gsk_`)
- No extra spaces before/after the key
- Key must be active (not deleted from Groq console)

### "Rate Limit" Error
- Free tier has limits (usually very generous)
- Wait a few minutes and try again
- Or create a new API key

### Key Expired
- Groq keys don't expire automatically
- But you can revoke/delete them from console
- Just create a new one if needed
